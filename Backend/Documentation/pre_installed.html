Overview

To streamline the development of custom filters on Filter Troy, we provide several commonly used Python libraries pre-imported in the environment. This convenience eliminates the need for users to include standard import statements in their filter code, allowing them to focus on implementing the filter logic directly.

Pre-Imported Libraries

The following libraries are available by default in the Filter Troy coding environment:

  1. NumPy (np)
    • Usage: NumPy is fundamental for scientific computing with Python. It provides support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.
    • Alias: np
    • Documentation: NumPy Documentation
  2. OpenCV (cv2)
    • Usage: OpenCV is an open-source computer vision and machine learning software library. It is useful for a wide range of applications such as face recognition, motion tracking, and image manipulation.
    • Alias: cv2
    • Documentation: OpenCv Documentation
  3. Pillow (Image from PIL)
    • Usage: Pillow is the Python Imaging Library (PIL) fork, providing capabilities for opening, manipulating, and saving many different image file formats.
    • Alias: Directly available via Image from the PIL module.
    • Documentation: Pillow Documentation
  4. Scikit-image (ski)
    • Usage: Scikit-image is a collection of algorithms for image processing in Python. It is built on top of SciPy and is designed to work with NumPy arrays, providing a comprehensive set of image processing functions.
    • Alias: ski
    • Documentation: Scikit-image Documentation

Conclusion

By providing these libraries, Filter Troy aims to reduce setup overhead and make it easier for users to focus on creating and customizing image filters. Enjoy the ease of using these powerful tools in your filter development!
